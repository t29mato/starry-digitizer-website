StarryDigitizer

Swiftly extract values from graph images
No installs needed, fully open source

StarryDigitizer is an open source value extraction tool that operates in a web browser.
Users can upload any graph image and swiftly and accurately extract data values using multiple automated extraction logics.
Since the images are not sent tothe server, there is no need to worry about rights-related concerns.

Features

Automatic Extraction
by Color

The system automatically extracts values from the graph based on the specified color.

Color options are presented to the user automatically, utilizing color information within the graph image.

There are two modes available: Line Extract mode, suitable for extracting line graphs, and Symbol Extract mode, ideal for extracting symbols in scatter plots.

Spline Interpolation

In cases where automatic extraction based on color is challenging, especially with complex graph images, if its a curved graph, you can use the spline interpolation feature.

This feature allows users to obtain values along the graph lines at any desired interval with minimal clicks.

During the operation, the interpolation lines are updated in real-time, providing a clear visual indication of which portions of the graph values can be extracted with just a glance.

Why "Starry"?

StarryDigitizer was originally developed as part of the web system of Starrydata project,
which is aimed at building an open database of inorganic materials science experimental data,
with the motivation to streamline the process of extracting graph data
and to obtain the most suitable data format for handling in the Starrydata system.

Our goal is not only to benefit Starrydata but also to develop
a valuable tool for anyone involved in collecting graph data.
